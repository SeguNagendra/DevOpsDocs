## Why a Maturity Model Matters

- Costs grow unpredictably

## Best Practices Across All Levels

### Tagging
- Consistent keys
- Unified service tagging
- Ownership tags

### Alerting
- Alert on impact
- Use SLOs
- Avoid noisy thresholds

### Dashboards
- Audience-specific
- Simple and actionable

### Cost
- Monitor usage
- Control cardinality
- Review regularly
